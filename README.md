# 项目简要说明

第一次尝试在esp32上移植lvgl。
没想到esp32官方提供的组件这么好用。

# 环境说明

## 1 软件环境

1. esp-idf: v5.3.4
2. 使用项目模版：sample_project

## 2 硬件环境

1. 使用正点原子 esp32s3M 最小系统板
2. LCD 屏幕: esp32s3M 最小系统板的自带小屏幕ST7735S，160\*80
